How to get your project live on the internet in 15 minutes (sorta)

Nowadays getting a web page up on the internet is almost trivial, in this small tutorial I will try to show how to do it courtesy of Google App Engine.

GAE is a framework/service that allows you to upload your app on Google servers, and it will take care of hosting it, provides a basic dns address, and makes updates painless.

They also provides a free daily quota that should be more than enough for any test/proof of concept web app. The idea is giving you a quick way of uploading your work on the internet.

I made a very basic skeleton for a web app (based on the examples in the excellent Web Application class from Steve Huffman on Udacity), which you can use as a template/base.

The backend is python based, but it is not necessary to modify it, for a very basic use.

At the very least you should be able to change the index.html (this very page) and the main.css with your own, and be able to upload a single page.

  • Step 1: Download and Install Python 2.7

    GAE supports different backend languages, I like python personally (and you should be very familiar with it if you took some CS class on Udacity). If you don't have it installed already (run python --version from the command line to check), you can download the latest version here:

    Python 2.7

  • Step 2: Download and Install Google App Engine SDK

    Make sure you select the version for your OS and the one based on Python.

    Google App Engine SDK

  • Step 3: Clone or download this same webpage from github

    You can find the skeleton for the application here:

    Download/clone the files in a local directory.

  • Step 4: Create an existing app in GAE pointing to the downloaded application

    • Run Google App Engine Launcher
    • Select "Add Existing Application" from the "File" menu
    • Browse to the path where you download the code, and press "Add"

  • Step 5: Change/edit the index.html and main.css files with your own code

  • Step 6: Deploy the application on Google servers

  • Step 7: Profit!
